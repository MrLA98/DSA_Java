#### 2.2 数组模拟环形队列

- 修改front和rear变量的含义 - 左闭右开` [front, rear)`
  - front变为队列的开始 -- 初始为0
  - rear变为队列结尾的下一个位置 -- 初始为0
- 队列满 -- `(rear + 1) % maxSize = front`
- 队列空 -- 不变：rear == front
- 队列的有效数据个数 ： `(rear + maxSize - front) % maxSize`
